IoT Based Baby Monitoring System

  • IoT, Python, Android Framework, MQTT

he proposed system exploits sensors to monitor the baby's vital parameters, such as ambient temperature, moisture, and crying. A prototype of the proposed baby cradle has been designed using Nx Siemens software, and a red meranti wood is used as the material for the cradle. The system architecture consists of a baby cradle that will automatically swing using a motor when the baby cries. Parents can also monitor their babies' condition through an external web camera and switch on the lullaby toy located on the baby cradle remotely via the MQTT server to entertain the baby.
